== Description ==

In simple way allows you to change colors of your website to grayscale using only few lines of CSS. You can use it for eg. mourning.
Plugin support browsers: Chrome 18+, Firefox 4+, IE 6+, Safari 5.2+, Opera 13+

== Installation ==

1. Upload plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How enable/disable plugin? =

This is very simple plugin so you only need to activate or deactivate it through the 'Plugins' menu in WordPress.
